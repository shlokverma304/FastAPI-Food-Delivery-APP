# Food Delivery Backend API (FastAPI)

># Project Overview
This project is a Food Delivery Backend API built using FastAPI.  
It includes menu management, cart system, order placement, checkout, search, sorting, and pagination.


?# Features
- Menu CRUD operations (Add, Update, Delete, View)
- Order Management System
- Cart System with Checkout
- Search functionality (case-insensitive)
- Sorting (price, name, category)
- Pagination
- Advanced browsing (filter + sort + pagination)


># Tech Stack
- Python
- FastAPI
- Uvicorn
- Pydantic


># How to Run
pip install -r requirements.txt
uvicorn main:app --reload
